## API Reference

#### Send Message

```http
  POST /api/messages
```

| Parameter      | Type     | Description                |
| :------------- | :------- | :------------------------- |
| `access_token` | `string` | **Required**. Access Token |
| `to`           | `string` | **Required**. The Receiver |

#### Create User

```http
  POST /auth/register
```

| Parameter  | Type     | Description            |
| :--------- | :------- | :--------------------- |
| `email`    | `string` | **Required**. Email    |
| `password` | `string` | **Required**. Password |
